# User Stories

This directory contains the user stories for the Airbnb-like application, translated from the use case diagram. Each story follows the format:

> As a [role], I want to [action] so that [benefit].

## Files Included

- `user-stories.md` — This file with detailed user stories.
- `README.md` — This documentation file explaining the purpose and structure of the stories.

## How to Use

1. Review the user stories to understand system requirements from a user perspective.
2. Use these stories to guide development tasks, prioritization, and test case creation.
